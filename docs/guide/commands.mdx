## Essencial

- `/ajuda`: Retorna um menu de ajuda
- `/motivos`: Retorna todos os motivcos válidos para uma denúncia
- `/reortar`: Denunciar um jogador
- `/jogar`: Retorna todos os comandos para conectar instantâneamente num minijogo
- `/j`: Retorna o selector de minijogos/servidores

## Conexão Rápida

- `/lobby`, `/parque` ou `/l` : Conecta instantâneamente ao Lobby
- `/bedwars`: Conecta instantâneamente ao BedWars
- `/skrush`: Conecta instantâneamente ao SkyRush
- `/skywars`: Conecta instantâneamente ao SkyWars

## Avançado

- `/party`: Comando principal para parties
- `/parkour`: Comando principal para parkour
- `/watch`: Telespectar uma partida de BedWars ou SkyWars de um jogador
- `/stats`: Retorna as estatísticas de BedWars ou SkyWars de um jogador
