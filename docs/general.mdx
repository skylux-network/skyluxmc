## Quem somos?
**A Skylux Network é um servidor de minijogos dedicado ao Minecraft. Foi criada inicialmente por [Renildo](https://github.com/rrenildopereiraa) e [Ricardo](https://github.com/Domingoss12) com o nome de Concordia onde mais tarde teve o seu nome alterado para o nome que temos hoje, Skylux Network!**

### Mini-jogos disponíveis:
No momento a Skylux Network tem apenas dois minijogos:

- BedWars
- SkyWars
