---
title: Regras
---

Para modelos de reportes por favor utiliza um destes motivos ao criar denúncias.

:::warning Modelos customizáveis
No momento a Skylux não oferece suporte a motivos para denúncias customizáveis, se nenhuma destas alternativas representar a tua denúncia entra em contacto conosco através do nosso [Discord](https://discord.gg/t4dK4nH78z).
::: 

### Exploração de Falhas
Qualquer exploração de falhas no jogo ou sistema.

```
/report <jogador> hacker
```

---

### Vantagem Injusta
Uso de cheats ou modificações que proporcionem vantagens injustas.

```
/report <jogador> cheater
```

---

### Comportamento Inadequado
Comportamento ofensivo, linguagem inadequada ou assédio. 

```
/report <jogador> ofensivo
```

---

### Propaganda
Promoção de outros servidores ou conteúdo não relacionado à Skylux.

```
/report <jogador> divulgador
```

---

### Identificação Inadequada
Uso de skins ou nicks inapropriados ou enganosos.

```
/report <jogador> inapropriado
```

---

### Spam/Flood
Envio mensagens repetitivas, irrelevantes ou em grande volume.

```
/report <jogador> spammer
```

---

### Equipa
Agrupamento de equipas para obter vantagens injustas.

```
/report <jogador> equipa
```

---

### Linguagem Inadequada
Uso de linguagem vulgar, discriminatória ou ofensiva.

```
/report <jogador> inapropriado
```
